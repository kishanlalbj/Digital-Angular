Full Stack Mastery (online)

An online, part time program designed for developers in billable projects to transform into digital full stack developer.

Online programs

These programs are focused on experiential learning in an online model with expert mentors working with participants through the program. The learning is in an environment that combines the convenience of online and anytime access with the intensity of scheduled mentoring.

Tech Stack

MEA/RN tech stacks

The JavaScript full-stack : NodeJS, AngularJS or ReactJS, ExpressJS, Seneca, MongoDB, Cassandra, Redis and/or Neo4J

Java microservices + Angular/React

The Java microservices full-stack : Microservices built with Spring with JavaScript clients built using AngularJS or ReactJS with MongoDB, Cassandra, Redis and/or Neo4J
