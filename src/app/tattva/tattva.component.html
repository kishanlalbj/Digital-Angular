TATTVA

Tattva enables construction of several market-facing solutions such as real-time logistics tracking with sensor-based data, shop floor monitoring, clickstream analytics for ecommerce applications etc.

Technologies

Tattva is build on the MERN stack i.e MongoDB, Express.JS, React.JS and Node.JS. The front end involves the usage of HTML 5, CSS 3 and Semantic UI.
